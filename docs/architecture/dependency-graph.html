External boundary

  • f2 — third-party Douyin SDK. Wrapped exclusively from UserService, PostService, and LivestreamService; never imported from routers.
  • boto3 — used only inside R2StorageService. Routers and other services touch R2 through that single facade.
  • fastapi / pydantic / pydantic-settings — the request/response and config layer. Schemas live exclusively in schemas/; settings in core/settings.py.
  • prometheus_client — exposes the /metrics ASGI app and counter/histogram families used by both main and R2StorageService.
